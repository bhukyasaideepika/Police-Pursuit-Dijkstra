# Police Pursuit Pathfinding using Dijkstra's Algorithm

## Overview
This project implements Dijkstra's Algorithm to determine the shortest path in a graph-based police pursuit scenario. The program models locations as graph vertices and roads as weighted edges to compute the minimum-cost path between two locations.

## Features
- Implements Dijkstra's shortest path algorithm
- Graph representation using adjacency lists
- Efficient shortest path computation
- Handles weighted graphs
- Tested with multiple graph inputs

## Technologies Used
- C++
- Data Structures & Algorithms
- Graph Algorithms
- Priority Queue (Min Heap)
